## Installation

### Option 1: Using pip (Recommended)

```bash
pip install my-little-buddy
my-little-buddy
```

### Option 2: From Source

1. Clone or download this repository
2. Navigate to the project directory
3. Install the package:

```bash
pip install .
```

4. Run the game:

```bash
my-little-buddy
```

### Option 3: Direct Python Execution

If you prefer not to install:

```bash
python main.py
```

## How to Play

1. Start with 1 free gacha roll and 50 Buddy Bucks
2. Use the "Gacha Adopt!" button to get your first pet
3. Care for your pet using the action buttons (Feed, Play, Clean, Sleep)
4. Watch your pet evolve as you maintain high stats
5. Earn rewards through mini-games and achievements
6. Unlock new themes as you progress

## Controls

- Quick Actions: Press Q (Feed), W (Play), E (Clean), R (Sleep)
- Dev Mode: Press Spacebar 8 times to enable fast evolution

## System Requirements

- Python 3.8 or higher
- Operating System: Windows, macOS, or Linux
- Tkinter (usually included with Python)

## Distribution

For users who don't have Python installed, executables are available for:

- Windows
- macOS
- Linux

Check the releases section for pre-built executables.

## Development

To contribute or modify the game:

1. Fork the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (Linux/macOS) or `venv\Scripts\activate` (Windows)
4. Install in development mode: `pip install -e .`
